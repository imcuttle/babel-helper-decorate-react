> Babel Helper for custom decorator for React Component

## Installation

```bash
npm install babel-helper-decorate-react
# or use yarn
yarn add babel-helper-decorate-react
```

## Usage

```javascript
import babelPluginDecorateReact = from 'babel-helper-decorate-react'
```
